# SOC Home Lab, Detection & Analysis 

## Overview 
This repository outlines a hands-on SOC (Security Operations Center) home lab focused on detecting, analyzing, and documenting common attack behaviors using endpoint and network telemetry. 

This lab aims to help you think and work like an entry-level SOC analyst. You will simulate attacks and confirm detections using real evidence. 

--- 

## Lab Environment 
- **Victim:** Windows 10 Virtual Machine
- **Attacker:** Kali Linux Virtual Machine
- **Network:** VirtualBox NAT Network (isolated)
- **Monitoring Tools:** Sysmon, Wireshark

--- 

## Skills Demonstrated 
- Network traffic analysis
- Endpoint log analysis (Sysmon)
- TCP/IP and port scanning detection
- Incident documentation and reporting
- MITRE ATT&CK mapping
- SOC-style investigation workflow
